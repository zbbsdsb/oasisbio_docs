    Ability Pool System

    Overview

    The Ability Pool system is a core feature of OasisBio that allows users to define, manage, and visualize the skills, traits, and abilities of their identities. It provides a structured way to represent the capabilities of an identity across different eras and worlds.

    Ability Types

    1. Custom Abilities

    • Description: User-defined abilities that are unique to a specific identity
    • Use Case: Personal skills, unique talents, specialized knowledge
    • Example: "Quantum Physics", "Dragon Taming", "Cooking"

    2. Official Abilities

    • Description: Pre-defined abilities provided by the platform
    • Use Case: Common skills, standardized abilities, widely recognized talents
    • Example: "English", "Basketball", "Programming"

    Ability Categories

    OasisBio provides a comprehensive categorization system for abilities:

    1. Languages

    • Examples: English, Spanish, Japanese, French, Mandarin
    • Level Description: Fluency level (beginner to native)

    2. Sports

    • Examples: Basketball, Football, Running, Swimming, Skateboarding
    • Level Description: Skill level (amateur to professional)

    3. Arts

    • Examples: Drawing, Photography, Filmmaking, Fashion Styling, Calligraphy
    • Level Description: Proficiency level (hobbyist to master)

    4. Music

    • Examples: Rap, Singing, Beatmaking, Guitar, Piano
    • Level Description: Skill level (beginner to virtuoso)

    5. Technology

    • Examples: Frontend Development, AI Prompting, Game Design, 3D Modeling, Data Analysis
    • Level Description: Expertise level (novice to expert)

    6. Science

    • Examples: Physics, Chemistry, Biology, Astronomy, Mathematics
    • Level Description: Knowledge level (basic to advanced)

    7. Social Skills

    • Examples: Public Speaking, Negotiation, Networking, Empathy, Leadership
    • Level Description: Proficiency level (beginner to master)

    8. Survival Skills

    • Examples: Camping, Navigation, First Aid, Foraging, Self-Defense
    • Level Description: Competency level (basic to expert)

    9. Worldbuilding Skills

    • Examples: Storytelling, Map Making, Character Creation, Lore Development
    • Level Description: Skill level (hobbyist to professional)

    10. Combat / Fantasy Skills

    • Examples: Swordsmanship, Magic, Martial Arts, Archery, Telepathy
    • Level Description: Power level (novice to master)

    11. Spiritual / Abstract Traits

    • Examples: Intuition, Creativity, Wisdom, Resilience, Empathy
    • Level Description: Strength level (weak to strong)

    12. Profession Tags

    • Examples: Engineer, Artist, Scientist, Teacher, Entrepreneur
    • Level Description: Professional level (entry to expert)

    Ability Structure

    Each ability includes the following attributes:

    • Name: The name of the ability
    • Category: The category the ability belongs to
    • Type: Custom or official
    • Level: Skill level (1-5)
    • Description: Detailed description of the ability
    • Related World: World the ability is associated with (optional)
    • Related Era: Era the ability is associated with (optional)
    • Visibility: Public or private

    Skill Levels

    OasisBio uses a 5-point scale to measure ability levels:

    Level Description Example
    1 Beginner Basic understanding, limited experience
    2 Intermediate Working knowledge, regular practice
    3 Advanced Proficient, consistent application
    4 Expert Deep knowledge, specialized skills
    5 Master Exceptional ability, recognized authority

    Ability Management

    Adding Abilities

    1. Select Category: Choose the appropriate category for the ability
    2. Enter Details: Provide name, description, and other attributes
    3. Set Level: Assign a skill level (1-5)
    4. Add Context: Link to specific worlds or eras if applicable
    5. Save: Add the ability to the pool

    Editing Abilities

    • Update ability details
    • Change skill level
    • Modify category
    • Update world or era associations
    • Adjust visibility settings

    Deleting Abilities

    • Remove abilities from the pool
    • Confirmation required before deletion

    Ability Visualization

    Tag Chips

    • Description: Small, colored chips representing each ability
    • Usage: Displayed in the ability pool section of the profile
    • Design: Color-coded by category, with level indicated by size or additional indicators

    Radar Charts

    • Description: Visual representation of ability levels across categories
    • Usage: Provides an at-a-glance view of an identity's strengths and weaknesses
    • Design: Multi-axis chart with categories as axes and level as distance from center

    Expandable Cards

    • Description: Detailed ability information displayed in expandable cards
    • Usage: Clicking on an ability tag opens a card with full details
    • Design: Clean, minimalist cards with ability information and level indicator

    Era and World Binding

    Era Binding

    • Description: Link abilities to specific time periods
    • Use Case: Track how abilities develop over time
    • Example: "Spanish (Level 1 - High School)" vs "Spanish (Level 4 - Professional)"

    World Binding

    • Description: Link abilities to specific fictional worlds
    • Use Case: Abilities that only exist in certain worlds
    • Example: "Magic" in a fantasy world, "Telepathy" in a sci-fi world

    Best Practices

    1. Balance: Create a balanced set of abilities that reflect the identity's strengths and weaknesses
    2. Consistency: Maintain consistent ability levels across related skills
    3. Context: Link abilities to appropriate eras and worlds
    4. Detail: Provide detailed descriptions to give context to each ability
    5. Organization: Use categories effectively to organize abilities

    Future Enhancements

    • Ability Progression Tracking: Visualize how abilities develop over time
    • Ability Combinations: Create compound abilities by combining multiple skills
    • AI-Assisted Ability Suggestions: Get recommendations based on identity type and context
    • Ability Marketplace: Share and import ability sets with other users
    • Ability Challenges: Set goals and track progress for skill development
